About
=====

browser is a module that implements a web browser interface to automate tasks
in python.

Main features
-------------

 * Lightweight
 * Threadsafe
 * Pythonic
 * Fully configurable

Download
--------

 * Tar/Zip from http://github.com/pointtonull/browser/archives/master
 * Using git: git clone git://github.com/pointtonull/browser.git
